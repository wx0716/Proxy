<a name="1">1.Aria2.sh</a>

脚本说明：Aria2c一键安装脚本，详情介绍可访问官网：https://aria2.github.io

**使用方法：**

```bash
wget -N --no-check-certificate https://raw.githubusercontent.com/wx0716/Proxy/master/Aria2.sh && chmod +x Aria2.sh && bash Aria2.sh
```
